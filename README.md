# Registration System Assignment

This project is developed as part of the Frugal Testing Software Engineer assignment.

## Project Overview
The Registration System is a web-based form designed using HTML, CSS, and JavaScript.  
It collects user details and validates inputs before submission to ensure correct and complete data.

## Features
- User registration form with mandatory fields
- Client-side validation with inline error messages
- Email validation to block disposable domains
- Password strength indicator (Weak / Medium / Strong)
- Submit button enabled only when all validations pass
- Success message displayed after successful submission

## Technologies Used
- HTML
- CSS
- JavaScript
